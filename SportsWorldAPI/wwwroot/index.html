SportsWorld API

Tennis backend-API made with ASP.NET and Enitity Framework Core

Athletes

Get all registered tennisplayers.

Available Endpoints:

GET /api/athlete - Get all athletes

GET /api/athlete/{id} - Get athlete by id

POST /api/athlete - Create athlete

PUT /api/athlete/{id} - Update athlete

DELETE /api/athlete/{id} - Remove athlete

Venues

Get all available venues.

Available Endpoints:

GET /api/venues - Get all venues

GET /api/venues/{id} - Get venue by id

POST /api/venues - Create venue

PUT /api/venues/{id} - Update venue

DELETE /api/venue/{id} - Remove venue

Finance

Get overview of finances to buy tennisplayers.

Available Endpoints:

GET /api/finance - Get current finances

POST /api/finance - Add loan

POST /api/finance/{athleteId} - Register purchase
